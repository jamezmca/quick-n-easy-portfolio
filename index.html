Galactiplanner

React | NoSQL | Firebase Authentication

Galactiplanner is a responsive web app that allows you to take your plans with you everything thanks to Firebase authentication and data hosting. You can login on your phone or computer and efficient data entry and planning system will help keep you organised.

Moonglade Apparel

React | Stripe.js | Commerce.js | Material UI | Netlify

Moonglade Apparel is a 'save the planet' inspired clothing brand where 25% of profits are donated to charity. Moonglade apparel fetches data from Commerce.js as a content CMS and uses Stripe.js for transaction handling and billing. Components are styled using Material UI.

5Do

React Native | Expo

React Native is a React Native app developed using expo. Release on the google playstore, 5Do helps to priortise your goals and objectives. The 5Do theme is inspired by The Burning Crusade and the name stems from the idea that a good todo list is maximum 5 items long.
